How Instagram & Meta Ads connect
Connecting links the account. To actually post, read post data, and push/manage ads, Viktoriia needs one execution backend, set on the server:
  • Official Meta API (recommended for real posting + full ad management): set META_APP_ID, META_APP_SECRET, META_EXECUTION_MODE=api. Requires a Meta Developer app with Instagram + Marketing API permissions.
  • Browser agent (works without API access): set BROWSER_AGENT_URL; a person signs in once and Viktoriia operates the authenticated session.
Read & Write must be enabled per card, and every publish/spend still asks for your approval.
